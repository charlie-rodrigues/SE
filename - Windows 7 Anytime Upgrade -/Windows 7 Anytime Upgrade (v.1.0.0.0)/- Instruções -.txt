: ...:: Instalação ::... :
--------------------------

1 - Desative a internet e execute o programa "Windows 7 Anytime Upgrade Keygen.exe".

2 - Clique em "Generate" para obter um serial válido para o upgrade do Windows,

3 - No menu iniciar digite em pesquisa "Anytime Upgrade", abra-o, inclua o serial e faça o Upgrade.

4 - Após o Upgrade é recomendado usar um ativador, como o Windows Loader, Chew Wga ou RemoveWAT 

5 - Pronto !! Feito o Upgrade e Ativado !!
